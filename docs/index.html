WinOptimize Script

Una herramienta de optimización para Windows que mejora el rendimiento y la limpieza del sistema con un solo clic.

Funciones Principales

  • Limpieza Profunda: Elimina archivos temporales de %TEMP% y C:\Windows\Temp.
  • Optimización de Arranque: Borra la caché de Prefetch para agilizar el inicio del sistema.
  • Visor de Eventos Limpio: Elimina todos los registros del Visor de Eventos de Windows.
  • Máximo Rendimiento: Cambia automáticamente el plan de energía al modo "Máximo Rendimiento".
  • Optimización Avanzada: Ejecuta el popular script de Chris Titus Tech para ajustes adicionales.
  • Descarga de QuickCPU: Abre el navegador para que descargues QuickCPU.

Instrucciones de Uso

  1. Descarga el archivo WinOptimize.bat desde el botón de abajo.
  2. Haz clic derecho sobre el archivo descargado y selecciona "Ejecutar como administrador".
  3. Sigue las instrucciones que aparecerán en la ventana de comandos.
  4. ¡Disfruta de un sistema más limpio y rápido!
